# P2.5a A 型策略診斷

此報告只用於研究，未更動每日候選與 `p2.11_v2` 紙上交易規則。

以 2016–2026 的 59,343 筆 D0 漲停事件檢查目前 A 型漏斗：

| 條件階段 | 筆數 |
| --- | ---: |
| 全部漲停事件 | 59,343 |
| 盤整至少 5 日 | 13,632 |
| ATR5 小於 ATR20 | 10,062 |
| 成交量不高於 20 日中位量 1.2 倍 | 201 |
| 通過既有 MACD／流動性等硬條件 | 31 |

結論：A 型樣本不足的主要瓶頸是「D0 量比 ≤ 1.2 倍」，不是盤整或 ATR 收斂條件。

研究候選池（尚未採用）將量比放寬到 1.5 倍、ATR5 允許不高於 ATR20 的 1.1 倍，其餘
硬條件不變，可得到 61 筆。樣本仍偏小，因此下一步必須先做獨立 outcome 與 walk-forward
驗證；不得直接納入每日候選或與 B 型混合宣稱有效。
