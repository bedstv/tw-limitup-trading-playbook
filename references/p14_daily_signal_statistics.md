# P1.4 日線可驗證訊號統計

目的：用 P1.3 的 D1／D2 daily outcome labels，先檢查策略條件是否具有可觀察的訊號價值。此階段仍不是分時成交回測，也不代表真實損益。

## 執行指令

共用資料層：`../tw-market-data`

```bash
python3 scripts/summarize_p14_daily_stats.py \
  --outcomes-tag p13_2016_2026_selected \
  --output-tag p14_2016_2026_selected
```

對照組：

```bash
python3 scripts/summarize_p14_daily_stats.py \
  --outcomes-tag p13_2016_2026_all \
  --output-tag p14_2016_2026_all
```

## 核心結果

| 指標 | V1 候選池 | 全漲停池 |
|---|---:|---:|
| 樣本數 | 9,900 | 59,256 |
| D1 blowoff 數 | 969 | 5,229 |
| D1 blowoff 比率 | 9.79% | 8.82% |
| Blowoff 後 5 日內曾重返警示價 | 52.01% | 45.09% |
| Blowoff 後曾跌破失效價 | 84.83% | 87.23% |
| 重返早於失效價 | 22.81% | 18.01% |
| 重返早於或同日於失效價 | 32.09% | 27.02% |

解讀：V1 候選池相對全漲停池確實有較好的重返信號；但若要求重返必須早於失效價，命中率約 22.8%，不能直接解讀成高勝率交易策略。

## 分群觀察

### 流動性

| 流動性 | 候選數 | D1 blowoff | 曾重返 | 重返早於失效 | 重返早於或同日失效 |
|---|---:|---:|---:|---:|---:|
| 5,000 張以上 | 8,423 | 825 | 52.24% | 22.18% | 31.64% |
| 2,000～4,999 張 | 1,477 | 144 | 50.69% | 26.39% | 34.72% |

低流動性 fallback 的乾淨重返率略高，但樣本較少，且真實滑價風險可能更高。

### D1 開盤跳空

| D1 gap bucket | 候選數 | D1 blowoff | 曾重返 | 重返早於失效 | 重返早於或同日失效 |
|---|---:|---:|---:|---:|---:|
| 0～3% | 4,506 | 523 | 60.99% | 28.11% | 40.34% |
| 3～5% | 1,752 | 243 | 44.03% | 17.70% | 25.51% |
| 5～7% | 888 | 103 | 41.75% | 13.59% | 19.42% |
| >7% | 1,081 | 100 | 35.00% | 17.00% | 18.00% |

V1 的「跳空超過 5% 不直接追價、超過 7% 拒絕追價」方向合理：跳空越高，重返早於失效的比率大致越差。

### D1 量能倍數

| D1 / D0 量比 | 候選數 | D1 blowoff | 曾重返 | 重返早於失效 | 重返早於或同日失效 |
|---|---:|---:|---:|---:|---:|
| 1.5～2x | 1,556 | 424 | 52.83% | 25.47% | 33.73% |
| 2～3x | 1,265 | 386 | 51.30% | 22.54% | 32.12% |
| >3x | 580 | 159 | 51.57% | 16.35% | 27.67% |

使用 1.5～2x 作為觀察門檻合理；超過 3x 反而較差，可能代表籌碼釋放更重。

## 重要發現：A 型樣本不足

P1.2／P1.3 的 selected 候選幾乎全部是 B 型：

- B 型：9,877 筆
- A 型：23 筆
- D1 blowoff 中 B 型：965 筆
- D1 blowoff 中 A 型：4 筆

這表示目前 V1 硬條件實際上偏向「突破短前高帶量」策略；「盤整一週以上且不帶量」A 型樣本太少，尚不能統計有效性。若 A 型是策略核心之一，下一輪應調整 A 型定義或候選補入規則。

## 結論

日線統計支持「V1 候選池比全漲停池更有訊號」，尤其 D1 開高走低爆量後的 D2+ 重返警示價機率較高。但這個優勢在加入失效價條件後明顯收斂：

- 寬鬆看：52.0% 曾重返警示價。
- 嚴格看：22.8% 重返早於失效。
- 放寬同日排序不明：32.1%。

因此下一步不應直接進行獲利宣稱，而是做 P1.5：

1. 建立保守日線事件回測。
2. 對同日重返與失效採保守規則：同日視為先失效，或列為 ambiguous。
3. 設定明確進場價、停損價、出場 horizon。
4. 加入單邊交易成本 10～30 bps 敏感度。
5. 分群回測 0～3% gap、1.5～2x 量比等較佳子集合。
