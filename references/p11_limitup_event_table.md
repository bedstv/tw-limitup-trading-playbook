## 漲停價判定

### TWSE

使用官方每日收盤行情的收盤價、漲跌方向及漲跌價差還原當日開盤競價基準，再依
TWSE 官方升降單位向不超過 10% 的合法價位重建漲停價。不能只用
`close / previous_close >= 1.095` 近似。

### TPEx

使用前一交易日官方歷史行情直接公告的「次日漲停價／次日跌停價」，於下一交易日
與收盤價比對。除權、除息欄位即使不是數字，也不影響前一日公告漲停價的驗證。

### 無漲跌幅例外

依主檔上市櫃日期排除新上市首五個交易日。缺少前一交易日官方限制價格、競價基準
或必要歷史時，不得建立已驗證事件。
